Title:
Port Security Violation Investigation Guide

Purpose:
Investigate and resolve switch port security violations.

Symptoms:

* Port shutdown
* Unauthorized device detection
* Endpoint connectivity loss
* Security alerts

Investigation Steps:

1. Review port security status.
   Command:
   show port-security

2. Check MAC address table.
   Command:
   show mac address-table

3. Identify connected devices.

4. Review security policy configuration.

5. Verify authorized MAC addresses.

Root Cause Indicators:

* Unauthorized device connection
* MAC address limit exceeded
* Sticky MAC mismatch
* Unauthorized switch installation

Recommended Resolution:

1. Disconnect unauthorized devices.
2. Update security policies.
3. Clear incorrect MAC entries.
4. Restore port operation.

Expected Outcome:
Secure network access and restored connectivity.
